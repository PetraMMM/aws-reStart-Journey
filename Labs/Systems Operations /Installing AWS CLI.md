# AWS CLI Installation and Configuration on Red Hat Linux

## Project Overview
In this project, I installed and configured the AWS Command Line Interface (AWS CLI) on a Red Hat Linux EC2 instance. I intentionally used Red Hat because it doesn't come with the CLI pre-installed, giving me hands-on experience with the complete setup process.

## Key Achievements
* **Secure Access:** Established an SSH connection to my Red Hat Linux instance.
* **Installation:** Downloaded and installed the AWS CLI v2 manually.
* **Configuration:** Secured access to my AWS account by configuring my Access Key ID and Secret Access Key using `aws configure`.

<img width="500" height="230" alt="image" src="https://github.com/user-attachments/assets/9838a6bc-67d4-44fc-8b0f-44bd3e1ed706" />

* **Verification:** Successfully executed AWS CLI commands to interact with AWS Identity and Access Management (IAM) and validate my permissions.

## Technologies Used
* **Cloud Compute:** Amazon EC2 (Red Hat Linux)
* **Tools:** AWS CLI v2, SSH
* **Security:** AWS IAM, Programmatic Access Keys

## Conclusion
This lab gave me practical experience in server administration and programmatic AWS access.
